Power Commons  
==============

Power Commons is a useful Java Utils Library.

Power Commons 是非常有用的Java类库集合




## 项目信息 ##

- Java项目(1.7+)
- Maven管理(3.0.5+)

## 它是什么? ##

- 命名为Power Commons
- 高效、简单的 Java类库集合

## Features ##

支持的模块有：


- commons-jredis
- commons-lang
    - lang
    - config
    - data
    - io

## 使用 ##

在您的 Maven POM 文件里加入：

    <dependency>
        <groupId>org.power.commons</groupId>
        <artifactId>commons-lang</artifactId>
        <version>1.0.0-SNAPSHOT</version>
    </dependency>
        
    <dependency>
         <groupId>org.power.commons</groupId>
         <artifactId>commons-jredis</artifactId>
         <version>1.0.0-SNAPSHOT</version>
    </dependency>

主要依赖为：

- gson(2.3)
- guava(18.0)
